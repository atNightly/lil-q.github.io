位运算就是基于整数的二进制表示进行的运算。由于计算机内部就是以二进制来存储数据，位运算非常高效。

常用的运算符共 6 种，分别为与（ & ）、或（ | ）、异或（ ^ ）、取反（ ~ ）、左移（ << ）和右移（ >> ），java中有带符号位右移（ >>> )。

位运算的应用

位运算一般有三种作用：

  1. 高效地进行某些运算，代替其它低效的方式。
  2. 表示集合。（常用于 状压 DP 。）
  3. 题目本来就要求进行位运算。

需要注意的是，用位运算代替其它运算方式（即第一种应用）在很多时候并不能带来太大的优化，反而会使代码变得复杂，使用时需要斟酌。（但像“乘 2 的非负整数次幂”和“除以 2 的非负整数次幂”就最好使用位运算，因为此时使用位运算可以优化复杂度。）

leetcode 136. 只出现一次的数字

给定一个非空整数数组，除了某个元素只出现一次以外，其余每个元素均出现两次。找出那个只出现了一次的元素。

示例:

输入: [2,2,1]
输出: 1

利用异或运算的特性：x ^ 0 = x ， x ^ x = 0，对所有数做一遍异或就可以提出唯一数。

class Solution {
    public int singleNumber(int[] nums) {
        int res = 0;
        for (int num : nums) {
            res ^= num;
        }
        return res;
    }
}

leetcode 137. 只出现一次的数字 II

给定一个非空整数数组，除了某个元素只出现一次以外，其余每个元素均出现了三次。找出那个只出现了一次的元素。

示例:

输入: [2,2,3,2]
输出: 3

同样利用异或位运算来处理，由于位运算的特性，处理一位可以推导出处理一个数。如果是 0 ，不处理；如果是 1 ，有三种情况，相加求余分别为 0 ，1 ，2 。建立有限状态机：

class Solution {
    public int singleNumber(int[] nums) {
        int zero = ~0, one = 0, two = 0; // 表示余0，余1，余2的三种状态，每一位上1表示在该状态，0表示不在该状态
        for (int num : nums) {
            one = ~two & (one ^ num); // 余1状态要处理除了在余2状态的位，即 ~two & ... 
            two = ~zero & (two ^ num); // 余2状态要处理除了在余0状态的位，即 ~zreo & ...
            zero = ~(one | two); // 任意位都在三种状态中，最后更新zero
        }
        return one; 
    }
}

实际上还可以再简化过程，zero可以去掉，用已经更新过的one来更新two：

class Solution {
    public int singleNumber(int[] nums) {
        int one = 0, two = 0;
        for (int num : nums) {
            one = ~two & (one ^ num);
            two = ~one & (two ^ num);
        }
        return one; 
    }
}

leetcode 260. 只出现一次的数字 III

给定一个整数数组 nums，其中恰好有两个元素只出现一次，其余所有元素均出现两次。 找出只出现一次的那两个元素。

示例 :

输入: [1,2,1,3,2,5]
输出: [3,5]

求出两个数的异或，再分组处理。

class Solution {
    public int[] singleNumber(int[] nums) {
        int c = 0; // c = a ^ b
        for (int num : nums) c ^= num;
        int diff = c & -c; // 求最右边的第一个1
        int a = 0;
        for (int num : nums) if ((num & diff) == 0) a ^= num;
        return new int[] {a, c ^ a}; // 利用b ^ a ^ a == b
    }
}

leetcode 169. 多数元素

给定一个大小为 n 的数组，找到其中的多数元素。多数元素是指在数组中出现次数大于 ⌊ n/2 ⌋ 的元素。

你可以假设数组是非空的，并且给定的数组总是存在多数元素。

示例:

输入: [3,2,3]
输出: 3

可以按位求多数位：

class Solution {
    public int majorityElement(int[] nums) {
        int res = 0;
        for (int k = 0; k < 32; k++) {
            int count = 0;
            for (int num : nums) {
                count += (num >> k) & 1; // 计1的个数
            }
            if (count > nums.length >> 1) res += 1 << k;
        }
        return res;
    }
}

也可用Boyer-Moore 投票算法：

class Solution {
    public int majorityElement(int[] nums) {
        int maj = 0, count = 0;
        for (int num : nums) {
            if (count == 0) maj = num;
            if (num == maj) count++;
            else count--; 
        }
        return maj;
    }
}

leetcode 9. 回文数

这一题可以看出十进制的位运算：

class Solution {
    public boolean isPalindrome(int x) {
        if (x == 0) return true;
        if (x < 0 || x % 10 == 0) return false; // 除 0 以外以 0 结尾的都不是回文数
        int reversed = 0;
        while (x > reversed) {
            reversed = reversed * 10 + x % 10;
            x /= 10;
        }
        return x == reversed || x == reversed / 10; // 注意有奇数位和偶数维两个情况
    }
}

剑指offer 面试题65. 不用加减乘除做加法

写一个函数，求两个整数之和，要求在函数体内不得使用 “+”、“-”、“*”、“/” 四则运算符号。

class Solution {
    public int add(int a, int b) {
        while (b != 0) {
            int f = (a & b) << 1; // 计算进位
            a = a ^ b;
            b = f;
        }
        return a; 
    }
}

剑指offer 面试题15. 二进制中1的个数

本题中输入视为无符号数。

// 解一：
public class Solution {
    public int hammingWeight(int n) {
        int res = 0;
        while (n != 0) {
            res += n & 1; // 只计算一位
            n >>>= 1; // >>> 符号位一起移动，>> 符号位不移动
        }
        return res;
    }
}
// 解二：巧用n&(n−1)
public class Solution {
    public int hammingWeight(int n) {
        int res = 0;
        while (n != 0) {
            res++;
            n &= n - 1;
        }
        return res;
    }
}

剑指offer 面试题16. 数值的整数次方

经典快速幂：

class Solution {
    public double myPow(double x, int n) {
        if (n == 0) return 1;
        double p = 1;
        long m = n; // int最小值转为正数会溢出，需要转为long
        if (m < 0) {
            x = 1 / x;
            m = -m;
        }
        while (m > 0) {
            if ((m & 1) == 1) p *= x;
            x *= x;
            m >>>= 1;
        }
        return p;
    }
}
